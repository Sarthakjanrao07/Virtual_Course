# Virtual Course LMS 🚀

A modern, full-stack Learning Management System with AI-powered search and secure payments.

## 📂 Project Structure
- `/frontend`: React + Vite + Tailwind CSS
- `/backend`: Node.js + Express + MongoDB

## 🛠️ Tech Stack
- **Frontend**: React, Redux Toolkit, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, Mongoose, JWT
- **AI**: Google Gemini API
- **Payments**: Razorpay
- **Storage**: Cloudinary

## 🚀 How to Run Locally

### 1. Setup Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file with your credentials.
4. `npm run dev`

### 2. Setup Frontend
1. `cd frontend`
2. `npm install`
3. Create a `.env` file.
4. `npm run dev`

## 🌍 Deployment
The project is configured for **Vercel** deployment with dynamic CORS and API routing.
